Entendendo a Comunicação Cliente X Server

(Voltar)

A comunicação entre cliente e servidor é um aspecto essencial em ambientes computacionais distribuídos, onde sistemas interagem por meio de solicitações e respostas. No cenário web, por exemplo, o Hypertext Transfer Protocol (HTTP) é comumente empregado para essa interação. O cliente inicia a comunicação enviando uma requisição ao servidor, contendo informações sobre a ação desejada, como recuperar uma página ou submeter um formulário. O servidor, por sua vez, processa a requisição e envia uma resposta, que pode conter dados solicitados ou indicar o sucesso da operação. Essa comunicação é vital para a funcionalidade de aplicações online, possibilitando desde a navegação fluida em páginas web até transações seguras em comércio eletrônico.

Temos 2 lados quando se trata de uma aplicação WEB, lado do cliente e lado do servidor. Do lado do servidor temos algumas linguagens que são utilizadas como PHP, Ruby, c# e Java. Já do lado do cliente temos HTML (lembrando que HTML NÃO É LINGUAGEM DE PROGRAMAÇÃO), CSS e JavaScript. O que faz a conexão entre esses dois lados são os servidores.

Intodução ao HTML

(Voltar)

Tags

  • html: Representa a raiz do documento, serve com um container que engloba todos os outros elementos HTML;

  • body: É onde fica todo o conteúdo de texto, imagem e vídeos, em que o usuário vê e interage, nele os conteúdos são estruturados pelas demais tags do HTML;

  • head: Compreende as informações do documento que serão interpretadas pelo navegador (metadados). Como por exemplo, título do documento, links para folhas de estilo etc;

  • h1: É a tag para criar o título principal da página;

  • p: É a tag utilizada para atribuir parágrafos à página;

  • h2: É a tag para criar um subtítulo;

  • strong: Tag para negritar o texto;

  • i: Tag para formatar o texto em itálico;

  • img: Tag para adicionar uma imagem à página;

  • input: Cria uma caixa de texto para o usuário preencher.

Atributos de tags

(Voltar)

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc…

Exemplos de atributos

  • Atributo class:Permite atribuir uma ou mais classes a um elemento, facilitando a aplicação de estilos CSS específicos;

  • Atributo id: Atribui uma identificação única ao elemento, útil para estilos CSS e referências em scripts JavaScript;

  • Atributo style: Define estilos diretamente no elemento, como cor de texto, tamanho da fonte, etc;

  • Atributo src: Especifica a origem (URL) de recursos incorporados, como imagens, vídeos ou scripts;

  • Atributo href: Indica o destino de um link. Pode apontar para outra página, um documento, ou recurso online;

  • Atributo target: Especifica onde abrir o link. "_blank" abre em uma nova janela ou guia.
